## 🚀 **How to Launch**

### **Option 1: Modern Launcher (Recommended)**
```bash
python3 launch_modern_dashboard.py
```

### **Option 2: Direct Streamlit Run**
```bash
streamlit run streamlit_matrix_maximizer.py --server.port 8501
```

### **Option 3: With Custom Theme**
```bash
streamlit run streamlit_matrix_maximizer.py \
  --theme.base dark \
  --theme.primaryColor "#00d4ff" \
  --server.port 8501
```
