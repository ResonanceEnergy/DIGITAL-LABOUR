# MATRIX MAXIMIZER

> **Project Management & Intelligence Dashboard**
> Track all 27 ResonanceEnergy repositories, tasks, and progress

## Quick Start

```bash
# Streamlit Dashboard (PRIMARY)
cd MATRIX_MAXIMIZER
streamlit run dashboard.py

# Flask Alternative
python flask_dashboard.py
```

## Files

| File | Purpose |
|------|---------|
| `dashboard.py` | Streamlit-based dashboard with 2026 UI/UX, 3D visualizations, real-time metrics |
| `flask_dashboard.py` | Flask REST API + WebSocket version for integration |
| `launch.py` | Simple launcher script |

## Features

- **Portfolio Tracking**: All 27 repos with real progress calculations
- **Task Generation**: Automated task creation for OPTIMUS/GASKET
- **3D Network Visualization**: Interactive project relationship graphs
- **Real-Time Metrics**: CPU, memory, system health
- **AI Forecasting**: Predictive completion estimates
