# MATRIX MONITOR

**System Health & Performance Monitoring**

## Quick Start

```bash
cd MATRIX_MONITOR
python dashboard.py           # Full Flask dashboard
python metrics_collector.py   # Metrics only
python performance_validator.py  # 24-48hr validation
```

## Files

- `dashboard.py` - Full monitoring dashboard (Flask)
- `agent_monitor.py` - Inner Council agent tracking
- `metrics_collector.py` - Prometheus-style metrics
- `performance_validator.py` - Extended validation
- `panels/` - iOS panels for Pocket Pulsar
