Zero-friction capture for the BIT RAGE LABOUR second brain.

## When the User Texts Anything to Remember

Ideas, links, books, reminders, decisions, observations — capture immediately:

1. **Write to daily log**: `memory/YYYY-MM-DD.md` with today's date
2. **For durable facts** (preferences, decisions, identities): also append to `MEMORY.md`
3. **Confirm** capture with a brief acknowledgment

## Capture Guidelines
- **No folders, no tags, no complex organization**
- Just text and search
- Timestamp every entry
- Include context (who said it, where, why it matters)
- For links, include a brief description of what/why

## The Power Is in ZERO FRICTION
Capture as easy as texting. The user should never think twice about whether
to save something — the answer is always yes, instantly.

## Search
When the user asks "what did I save about X" or "find that link about Y":
- Use `memory_search` with their query
- Return the most relevant entries with timestamps
- If multiple matches, show them ranked by relevance
