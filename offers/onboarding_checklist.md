# Client Onboarding Checklist

## Before First Delivery

### From Client (Required)
- [ ] Company name
- [ ] What they sell (1–3 sentence product description)
- [ ] Target roles (e.g. "VP Sales", "CTO", "Head of Growth")
- [ ] Target company list OR ICP description (industry, size, geography)
- [ ] Preferred tone: neutral / casual / direct

### From Client (Optional but Improves Quality)
- [ ] Previous outreach examples (what worked / what didn't)
- [ ] CRM format preference (CSV columns, JSON fields)
- [ ] Knowledge base / help docs (for Support agent)
- [ ] Refund / escalation policies (for Support agent)
- [ ] Banned topics or competitors to avoid mentioning

### From Us (Setup)
- [ ] Confirm package and pricing
- [ ] Send invoice / payment link
- [ ] Confirm delivery schedule (weekly / daily)
- [ ] Confirm delivery method (email / shared folder / API)
- [ ] Set up client in billing tracker
- [ ] Run 2-3 test outputs and share for approval before full run
- [ ] Adjust prompts based on client feedback

### After First Delivery
- [ ] Collect feedback (what to keep, what to change)
- [ ] Tune personalization depth
- [ ] Confirm ongoing schedule
- [ ] Set reminder for Week 2 check-in
