FAQ

Common Questions

Everything you need to know before getting started.

How is this different from ChatGPT?
ChatGPT is a general chat interface. BIT RAGE SYSTEMS runs purpose-built multi-agent pipelines — research, generate, QA verify — that produce structured, ready-to-use outputs. Every task goes through quality gates. You get JSON/CSV outputs, not chat responses.
What LLMs do you use?
We use four providers — OpenAI GPT-4o, Anthropic Claude, Google Gemini, and Grok — with automatic failover. Each agent selects the best model for its task stage. You never worry about model selection.
How fast is delivery?
Average delivery is under 14 seconds per task. Sales outreach takes 12-15s. Support resolution averages 9.6s. Results are delivered via webhook callback, API polling, or email.
Can I integrate this into my existing tools?
Yes. We provide a REST API (POST /tasks) with webhook callbacks. Send a task, get structured JSON/CSV back. Works with any CRM, helpdesk, or automation tool. API key included with every account. Email us for API docs.
Is there a free trial?
Yes. Every new account gets 2 free sales leads (full enrichment + 3-email sequence) and 5 free support ticket resolutions. No credit card required to start. Email us to claim yours.
What about data privacy?
Your data is processed in real-time and not stored after delivery. We don't train models on your data. All API communications are encrypted via TLS. Zero data retention policy.
Do you offer custom agents?
Yes. If you need a specialized agent for your workflow — custom research, domain-specific extraction, or unique output formats — email us for a custom build quote.
Who built this?
BIT RAGE SYSTEMS is built by Resonance Energy — a Canada-based AI company. We build production AI systems, not prototypes. Get in touch.
